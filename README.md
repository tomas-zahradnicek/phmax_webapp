# PHmax kalkulačka ZŠ

## Spuštění
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Struktura
- `src/phmax-zs-logic.ts` – tabulky, typy, helpery, export CSV
- `src/phmax-zs-ui.tsx` – malé UI komponenty
- `src/App.tsx` – hlavní aplikace
- `src/styles.css` – jednoduché styly
